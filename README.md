# rukkiecodes

Built with [FusionUI](https://rukkiecodes.github.io/fusionui/) — **Nuxt** (Server-rendered or statically generated, file-based routing).

## Run it

```sh
npm install
npm run dev
```

## What's included

- `pinia`
- `vueuse`
- `shaders`
- `vitest`
- `eslint`

## Working with FusionUI

```sh
npx fusionui add f-glass       # add a component (--copy to vendor its source)
npx fusionui theme brand --primary=#195bff
```

Docs: https://rukkiecodes.github.io/fusionui/
